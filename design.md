# Brosa.ai Website Design Style Guide

## Design Philosophy

### Color Palette
- **Primary**: Teal (#14B8A6) - Modern, trustworthy, tech-forward
- **Secondary**: Deep Blue (#0EA5E9) - Professional, reliable, digital
- **Accent**: Soft Purple (#8B5CF6) - Creative, innovative, premium
- **Neutral**: Warm Gray (#64748B) - Sophisticated, readable, balanced
- **Background**: Pure White (#FFFFFF) - Clean, spacious, modern
- **Success**: Emerald (#10B981) - Positive, achievement, growth

### Typography
- **Display Font**: "Inter" - Bold, modern sans-serif for headings
- **Body Font**: "Inter" - Clean, readable for all content
- **Accent Font**: "JetBrains Mono" - Monospace for technical elements

### Visual Language
- **Minimalist Aesthetic**: Clean lines, generous white space, focused content
- **Gen Z Appeal**: Bold gradients, smooth animations, interactive elements
- **Professional Trust**: Consistent spacing, clear hierarchy, quality imagery
- **Tech Innovation**: Subtle glows, floating elements, modern iconography

## Visual Effects & Animations

### Core Libraries Used
1. **Anime.js** - Smooth micro-interactions and element animations
2. **Typed.js** - Typewriter effects for dynamic text
3. **Splitting.js** - Advanced text animations and reveals
4. **ECharts.js** - Interactive data visualizations
5. **Splide.js** - Smooth carousels and sliders
6. **p5.js** - Creative coding for background effects
7. **Pixi.js** - High-performance visual effects

### Animation Strategy

#### Text Effects
- **Typewriter Animation**: Main headlines type out character by character
- **Gradient Cycling**: Headlines cycle through teal-to-blue gradient colors
- **Split Letter Stagger**: Individual letters animate in with staggered timing
- **Highlight Pulse**: Key phrases pulse with subtle glow effects

#### Background Effects
- **Liquid Gradient Flow**: Animated gradient shapes that slowly morph and flow
- **Floating Particles**: Subtle particles that respond to mouse movement
- **Geometric Patterns**: Rotating geometric elements with parallax depth
- **Aurora Gradient**: Soft, flowing gradient background with color transitions

#### Interactive Elements
- **3D Button Lifts**: Buttons lift with shadow expansion and gradient shifts
- **Card Hover States**: Cards tilt and lift with smooth shadow transitions
- **Image Reveals**: Images fade in with scale and opacity transitions
- **Form Field Focus**: Input fields expand with animated border highlights

### Header Effects
- **Sticky Navigation**: Navigation bar adapts with blur background on scroll
- **Logo Animation**: Logo scales and glows on hover
- **Menu Transitions**: Smooth slide-in mobile menu with staggered items
- **Active States**: Animated underlines for navigation items

### Scroll Animations
- **Fade-in Reveals**: Content fades in as it enters viewport
- **Slide-up Stagger**: Elements slide up with staggered timing
- **Counter Animations**: Numbers count up when scrolled into view
- **Progress Indicators**: Animated progress bars for statistics

### Micro-interactions
- **Loading States**: Smooth skeleton loading with shimmer effects
- **Success Animations**: Checkmarks with bounce and scale effects
- **Hover Feedback**: Immediate visual feedback on all interactive elements
- **State Transitions**: Smooth transitions between different states

## Layout & Spacing

### Grid System
- **Container Max-width**: 1200px for optimal readability
- **Grid Columns**: 12-column flexible grid system
- **Breakpoints**: Mobile-first responsive design
- **Gutters**: 24px spacing between columns

### Spacing Scale
- **XS**: 4px - Fine details, borders
- **SM**: 8px - Tight spacing, icons
- **MD**: 16px - Standard spacing, padding
- **LG**: 24px - Section spacing, margins
- **XL**: 48px - Major section breaks
- **XXL**: 96px - Hero section spacing

## Component Styling

### Buttons
- **Primary**: Teal background, white text, subtle shadow
- **Secondary**: White background, teal border and text
- **Hover**: 3D lift effect with gradient shift and glow
- **Active**: Pressed state with reduced shadow

### Cards
- **Background**: White with subtle border
- **Shadow**: Soft drop shadow with hover expansion
- **Border Radius**: 12px for modern, friendly appearance
- **Hover**: Lift and tilt with enhanced shadow

### Forms
- **Input Fields**: Clean borders with focus animations
- **Labels**: Floating labels that animate on focus
- **Validation**: Smooth error/success state transitions
- **Buttons**: Consistent with global button styling

## Responsive Design

### Breakpoints
- **Mobile**: 320px - 768px
- **Tablet**: 768px - 1024px
- **Desktop**: 1024px - 1200px
- **Large**: 1200px+

### Mobile Optimizations
- **Touch Targets**: Minimum 44px for accessibility
- **Typography**: Larger text sizes for mobile readability
- **Spacing**: Reduced spacing for smaller screens
- **Navigation**: Slide-in menu for mobile

## Accessibility

### Color Contrast
- **Text on White**: Minimum 4.5:1 contrast ratio
- **Interactive Elements**: Clear focus indicators
- **Status Colors**: Red for errors, green for success

### Motion Preferences
- **Reduced Motion**: Respect user preferences for reduced animations
- **Fallback States**: Static alternatives for all animated content
- **Focus Management**: Proper keyboard navigation support

This design system creates a modern, engaging experience that appeals to Gen Z users while maintaining professional credibility for business customers. The combination of smooth animations, clean typography, and strategic use of color creates a memorable and effective user interface.
